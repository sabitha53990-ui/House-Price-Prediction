# House-Price-Prediction

## Project Overview
This project predicts house prices using the Linear Regression algorithm based on features such as bedrooms, bathrooms, floors, and year built.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Results
The model was trained and tested on housing data. A scatter plot of Actual vs Predicted prices showed good prediction performance.

## Conclusion
This project demonstrates how Machine Learning can be used to predict house prices and support data-driven decision-making.
